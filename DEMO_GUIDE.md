# 🎬 Demo 演示指南

本指南幫助您錄製精美的 Marine AI Assistant 展示影片。

## 🎯 推薦演示流程（3-5 分鐘）

### 第 1 幕：首頁介紹 (30 秒)

1. **開啟應用**
   - 展示首頁英雄區塊
   - 滑動展示三大功能卡
   - 展示熱門地點區塊

2. **主題切換**
   - 點擊右上角切換深色/淺色主題
   - 展示主題轉換動畫

### 第 2 幕：AI 對話（重點）(90 秒)

1. **進入對話頁**
   - 點擊「開始對話」按鈕
   - 展示歡迎畫面

2. **快速提示**
   - 點擊快速提示：「幫我評估明天 06:00 在旗津是否適合新手衝浪」
   - 等待 AI 回覆與工具卡顯示

3. **展示工具卡**
   - 天氣卡：溫度、風速、降雨、能見度
   - 海況卡：浪高、週期、趨勢圖
   - 潮汐卡：當前潮位、高低潮時間

4. **手動輸入**
   - 輸入：「金山今天下午適合衝浪嗎？」
   - 展示打字動畫與 AI 思考動畫

5. **複製功能**
   - 滑鼠移到 AI 訊息上方
   - 點擊複製按鈕

### 第 3 幕：儀表板 (60 秒)

1. **進入儀表板**
   - 點擊導航列的「儀表板」

2. **展示 KPI**
   - 快速掃過 8 個 KPI 卡片
   - 指出警示訊息（如有）

3. **圖表互動**
   - 滑鼠移到圖表上展示 Tooltip
   - 展示 4 組圖表：
     - 風速/風向
     - 浪高/週期
     - 潮汐曲線
     - 降雨/能見度

4. **切換地點**
   - 點擊不同地點按鈕
   - 展示資料更新動畫

### 第 4 幕：地圖探索 (60 秒)

1. **進入地圖頁**
   - 點擊導航列的「地圖瀏覽」

2. **地圖互動**
   - 拖曳地圖
   - 縮放地圖
   - 點擊地標展示 Popup

3. **點位列表**
   - 在左側列表搜尋「金山」
   - 點擊列表項目，地圖自動跳轉

4. **收藏功能**
   - 點擊星號收藏地點
   - 展示星號填滿動畫

5. **快速查詢**
   - 點擊「查詢海況」
   - 自動跳轉到 AI 對話頁

### 第 5 幕：響應式展示 (30 秒)

1. **調整視窗**
   - 縮小瀏覽器視窗模擬手機
   - 展示底部導航列
   - 展示卡片堆疊布局

2. **結束畫面**
   - 回到首頁
   - 展示品牌與 Logo

## 📱 錄影技巧

### 瀏覽器設定
- 使用 Chrome 開發者工具的裝置模擬器
- F12 → 點擊手機圖標 → 選擇 iPhone 或 Pixel
- 建議解析度：1920x1080

### 錄影軟體推薦
- **Windows**: OBS Studio、Camtasia
- **Mac**: QuickTime、ScreenFlow
- **線上**: Loom

### 錄影設定
- 解析度：1080p
- 幀率：30 fps
- 格式：MP4

### 美化技巧
1. 隱藏瀏覽器書籤欄（Ctrl/Cmd + Shift + B）
2. 使用無痕模式避免擴充套件干擾
3. 清空快取確保載入速度
4. 準備好要輸入的文字（避免打字錯誤）
