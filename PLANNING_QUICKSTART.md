# 🗓️ 行程規劃功能快速開始

本指南幫助您在 5 分鐘內體驗完整的行程規劃功能。

## 🚀 立即體驗

### 方式 1：AI 對話規劃（最簡單）

1. **啟動專案**
   ```bash
   npm install
   npm run dev
   ```

2. **進入 Chat 頁面**
   - 訪問 `http://localhost:5173/chat`
   - 或點擊首頁的「開始對話」

3. **點擊快速提示或輸入**
   ```
   明天想在旗津安排一日輕旅行，機車移動、預算1500、想吃海鮮、一定要看日落
   ```

4. **查看結果**
   - 📅 完整行程時間軸
   - 📍 附近 POI 列表
   - ⚠️ 警示提醒
   - 💰 預算分析

### 方式 2：視覺化介面

1. **進入規劃頁面**
   - 訪問 `http://localhost:5173/plan`
   - 或點擊導航列的「行程規劃」

2. **設定參數**
   - 選擇地點：旗津/金山/都歷
   - 選擇日期：明天
   - 交通方式：機車 🛵
   - 預算：1500 元
   - 人數：2 人

3. **生成行程**
   - 點擊「生成行程」按鈕
   - 等待 1-2 秒

4. **查看與調整**
   - 查看完整時間軸
   - 瀏覽附近 POI
   - 回到 Chat 頁面調整

## 🐛 常見問題

### Q: 行程卡沒有顯示？
**A:** 確保輸入包含「安排」「規劃」「行程」等關鍵字，或提及地點名稱。

### Q: POI 列表是空的？
**A:** Mock 資料只包含旗津、金山、都歷。確保選擇這三個地點之一。

### Q: 時間軸顯示錯誤？
**A:** 檢查瀏覽器 Console 是否有錯誤。確保所有檔案都已正確安裝。

### Q: 如何新增更多 POI？
**A:** 編輯 `src/mocks/poi.ts`，按照現有格式新增資料。

### Q: 如何修改行程邏輯？
**A:** 編輯 `src/mocks/planningData.ts` 中的 `generateDayPlan` 函數。
